# 中文小说 Scene 边界标注指导书

## 一、什么是 Scene 边界

Scene（场景）是小说中一段**时空连续、叙事焦点一致**的叙事单元。

**Scene 边界 = 两个 Scene 之间的切分线。** 当读者需要在脑海中"切换画面"时，就是一个边界。

---

## 二、核心判断标准：一句话原则

> **如果两个相邻段落需要不同的画面来想象，就应该切 scene。**

反过来：如果你能用同一个画面想象两个段落，就不应该切。

---

## 三、必须切 Scene 的情况（强硬规则）

### 3.1 地点变化

人物所处的位置发生了明确变化。

**示例：**
```
[P1] 她推开门走进办公室，桌上的文件堆成了小山。
[P2] 半小时后，她坐在咖啡厅靠窗的位置，看着窗外的雨发呆。
```
→ **切在 P1 后**。办公室 → 咖啡厅，地点明确变化。

**注意：** 同一建筑内不同房间也算地点变化（卧室→客厅→厨房不算，卧室→公司办公室算）。

### 3.2 时间跳跃

时间发生了超过当前叙事节奏的跳跃。

**示例：**
```
[P1] 那一夜，他翻来覆去睡不着，脑子里全是白天的对话。
[P2] 第二天一早，阳光透过窗帘照进来时，他已经做出了决定。
```
→ **切在 P1 后**。"那一夜" → "第二天一早"，明确时间跳跃。

**注意：** 同一场景中"过了一会儿""片刻后"不算时间跳跃。时间标记词如"第二天""次日""当晚""几天后""三个月后"通常意味着切。

### 3.3 事件/情节切换

叙事从一段完整的动作/事件转入另一段不直接连续的动作/事件。

**示例：**
```
[P1] 两人在饭桌上激烈争吵，谁也不让谁。最后他摔了筷子起身离开。
[P2] 教室里，学生们正在安静地写作业。班主任站在讲台上扫视着每一个人。
```
→ **切在 P1 后**。饭桌争吵 → 教室场景，完全不同的画面和事件。

### 3.4 叙事焦点转移

叙述的"镜头"从一组人物或事件跳到了另一组。

**示例：**
```
[P1] 她站在窗前，看着他的车缓缓驶出小区。眼圈红了，但她没有哭。
[P2] 车里，他把音乐开到最大，试图盖住脑海中那些挥之不去的画面。
```
→ **切在 P1 后**。视角从"她"切换到"他"，两个不同的空间和人物焦点。

### 3.5 闪回/回忆进入或退出

当叙事跳入回忆或从回忆中返回时。

**示例：**
```
[P1] 她看着他递来的照片，上面是两个孩子的笑脸。
[P2] 十年前的那个夏天，她第一次见到这两个孩子。那时他们刚满三岁，走路还摇摇晃晃的。
```
→ **切在 P1 后**。从现在进入回忆。

### 3.6 元信息结束后恢复正文

作者注、章节标题、感谢名单等元信息结束后，正文恢复处应切。

**示例：**
```
[P1] 作者有话要说：最近生病了，更新有点慢，抱歉抱歉。
[P2] 第二天早上六点，闹钟准时响起。她睁开眼，发现自己竟然睡在了沙发上。
```
→ **切在 P1 后**。作者注 → 恢复正文。

## 四、绝对不切的情况（强硬规则）

### 4.1 同一场景的连续动作

```
[P1] 她拿起桌上的杯子，抿了一口咖啡。
[P2] 随后翻开文件夹，开始一页页地审阅合同。
```
→ **不切。** 同一人物，同一地点，连续动作。

### 4.2 同一人物内心独白的延续

```
[P1] 他不明白自己为什么会答应这个要求。也许是因为愧疚，也许是出于同情。
[P2] 但不管怎样，现在后悔已经来不及了。既然答应了，就只能硬着头皮去做。
```
→ **不切。** 同一人物的内心活动，连续思考。

### 4.3 同一想法的多段表述

```
[P1] 这个世界太大了。大到她每次抬头看星空，都会觉得自己渺小得像一粒尘埃。
[P2] 可有时候她又觉得，世界太小了。小到走过了千山万水，还是会在某个街角遇到不该遇到的人。
```
→ **不切。** 同一人物的同一段感慨，只是分成两段表达。

### 4.4 单纯的情绪变化

```
[P1] 她看着窗外的落花，突然难过起来。
[P2] 明明早上还好好的，怎么到了下午，一切都变得那么沉重。
```
→ **不切。** 只是情绪变化，时空和焦点没变。

### 4.5 环境描写的延续

```
[P1] 窗外是一片翠绿的竹林，风过处沙沙作响。
[P2] 竹林深处有一条石板小路，蜿蜒通往山脚下的村庄。
```
→ **不切。** 同一场景的环境描写，镜头只是稍微移动。

## 五、灰色地带处理原则

遇到难以判断的情况，按以下优先级：

1. **时空是否变了？** → 变了就切
2. **画面是否需要重新想象？** → 需要就切
3. **宁可少切，不要多切** — 保守策略
4. **如果不确定，不切**

---

## 六、标注输出格式

**输入：** 编号段落序列 `[P1] [P2] ... [PN]`

**输出：** 纯 JSON，只有 boundaries 字段：
```json
{
  "boundaries": [N]
}
```

- `boundaries=[3]` 表示在 P3 和 P4 之间切换 scene
- `boundaries=[]` 表示整段文本无需切 scene
- **不要输出 reasons、analysis 或任何解释文字**
- **不要输出 JSON 以外的任何内容**

---

## 七、完整标注示例

### 示例 1：简单两场景
```
段落：
[P1] 下课铃响，她把课本塞进书包。
[P2] 走廊里人声嘈杂，她低着头穿过人群。
[P3] 回家路上，她拐进那家常去的便利店。
[P4] 便利店的空调很足，她站在冷柜前犹豫了很久。

输出：
{"boundaries": [3]}
```
→ P1-P2 在学校走廊，P3-P4 从回家路上到便利店，P3 明显换了地点。

### 示例 2：连续叙述不切
```
段落：
[P1] 他坐在书房里，面前的电脑屏幕上是一份未完成的报告。
[P2] 手指在键盘上悬了很久，却一个字也没敲出来。
[P3] 脑子里全是下午那通电话的内容。
[P4] 电话那头的声音平静得让他害怕。

输出：
{"boundaries": []}
```
→ 从头到尾同一人物、同一地点、连续心理活动，不需要切。

### 示例 3：时间跳跃
```
段落：
[P1] 深夜两点，她终于关掉了电脑。明天再说吧，她想。
[P2] 闹钟响了三遍她才从床上爬起来。
[P3] 镜子里的人眼睛肿得像核桃。她叹了口气，拧开水龙头。
[P4] 挤地铁的时候，她接到了他的微信。

输出：
{"boundaries": [1]}
```
→ P1 是深夜入睡前，P2-P4 是第二天早上起床到上班，明确时间跳跃。P2-P4 虽然从家里到地铁，但属于同一连续的"早上出门"场景。

### 示例 4：事件切换
```
段落：
[P1] 谈判桌上，双方僵持了整整三个小时。
[P2] 最后对方代表合上文件夹，说了句"我们再考虑"就起身离开了。
[P3] 晚上，他一个人坐在办公室里反复回放白天的对话。
[P4] 每一个细节都让他觉得对方还有话没说出口。

输出：
{"boundaries": [2]}
```
→ P1-P2 是谈判场景，P3-P4 是晚上独处反思。地点变了、时间变了、氛围变了。

---

## 八、关键提醒

1. **13 段的文本通常只有 0-3 个边界**，不要过度切分
2. **大多数相邻段落都在同一个 scene 里**
3. **读完整段文本再标注**，不要边读边标
4. **保持一致性** — 同类型的边界信号（如地点变化）在所有样本中用同样标准
